## ライフプランシミュレーター (Family Finance Planner)

「ライフプランシミュレーター」は、現在の年齢から100歳までの「家族の資金繰り計画」を自動で推算・可視化するためのWebアプリケーションです。
ブラウザ上で動作し、専門的な知識がなくても、簡単な入力だけで精緻な将来の資産推移をシミュレーションすることができます。

## 🌟 主な機能

- **100歳までの精緻な資金計画**: 毎年の収入・支出・資産残高の推移をグラフと表で可視化。
- **税金・社会保険料の自動推計**: 額面年収から手取り額を自動計算。住宅ローン控除や退職金控除など複雑な計算にも対応。
- **インフレ・マクロ経済スライド対応**: 物価上昇（インフレ率）や年金のマクロ経済スライドなどを考慮した、よりリアルなシミュレーションが可能。
- **資産運用（NISA / iDeCo）のシミュレーション**: つみたてNISAやiDeCoを利用した運用益の計算、老後の計画的な取り崩し（Die with Zeroなど）のシミュレーションをサポート。
- **詳細なライフイベント設計**: 結婚・出産・教育費（公立/私立など進路別）、住宅購入・売却、車の購入・維持費、親の介護費用などを細かく設定可能。
- **シナリオ比較機能**: 「賃貸 vs 持ち家」「60歳退職 vs 65歳退職」など、異なる人生のシナリオをワンクリックで比較生成。
- **ローカル保存機能**: 入力したデータはブラウザ（localStorage）に保存されるため、何度でも見直しや再開が可能。

## 📂 ディレクトリ構成

- `index.html`: アプリケーションのメイン画面（UI構造）
- `style.css`: 画面のスタイリング・レスポンシブデザイン（モバイル対応など）
- `app.js`: シミュレーションのコア計算ロジック・グラフ描画・UI制御をおこなうJavaScript
- `usage_guide.webp`: 使い方を解説した動画（アニメーション）ファイル

## 🚀 使い方

1. 本プロジェクトのフォルダ内にある `index.html` をお使いのWebブラウザ（Chrome, Edge, Safariなど）で開きます。サーバーの立ち上げは不要です。
2. 「基本設定」「家族・住居」「仕事・老後」「資産運用」「ライフイベント」の各メニューから、想定するシナリオの数値を入力します。
3. 画面下部にグラフ（資産残高推移・キャッシュフロー）と詳細な年表（推移表）が自動で生成されます。
4. 画面上部の「使い方」ボタンから、簡単な使い方ガイド動画を確認することができます。

## 🛠 使用技術

- HTML5 / CSS3
- JavaScript (Vanilla JS - フレームワーク非依存)
- Chart.js (グラフ描画ライブラリ)
- html2canvas / jsPDF (結果のPDF・画像出力用)

## 🔒 セキュリティとプライバシー

入力した金融データや個人情報は、すべてお使いのブラウザ上のローカルストレージ（localStorage）にのみ保存されます。外部サーバーへの送信は一切行われないため、安心してご利用いただけます。

商用利用される場合はご相談ください。
